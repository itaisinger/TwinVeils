setup guide
i this guide ill guide you through setting up the project without external guidance exetrnal to this guide.

1. clone the main branch
2. go to servers\world server and run start.bat (or start.sh on linux). a cmd window will open, let it be.
3. the server now runs on your pc. open minecraft and add the server. The adress should be <your ip>:25565.
3.1. if you intend to play it singleplayer, use "localhost:25565" or the equivalant "127.0.0.1:localhost".
3.2 for multiplayer you should look up your ip, use some other guide (extenrdal guidance outside of this guide).

4. to add a player to the server, you must follow a few steps. open the server cmd window and do:

4.1 for singleplayer:
4.1.1. run: whitelist <your username>
4.1.2 find your UUID (search minecraft username to UUID), it should look like f946718e-31b0-439f-8eca-e4353741b2b6
4.1.3 run: lp user <UUID> parent set friends
4.1.3 run: team join friendsChat <UUID>

4.2 for multiplayer:


<details>
  <summary>the following section contains heavy spoilers. if you know nothing about this map and intend to play it single, dont read it. if you intend you play it multi and be a player yourself, unfortunately its not possible to do without spoilers.</summary>
  
  4.2.1 divide the players into two groups who dont know about each other.
  4.2.2 for every player in group 1 run:
  4.2.2.1 whitelist <username>
  4.2.2.2 lp user <UUID> parent set friends
  4.2.2.3 team join friendsChat <UUID>
  4.2.2 for every player in group 2 run:
  4.2.2.1 whitelist <username>
  4.2.2.2 lp user <UUID> parent set uni
  4.2.2.3 team join uniChat <UUID>
  this will make sure they are seperated in the chat, tab, spawns and such.
</details>


5. join the server and start playing!
